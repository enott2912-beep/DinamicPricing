# Алгоритм динамического ценообразования для ритейла (MVP)

Проект для генерации, анализа и применения алгоритмов динамического ценообразования для ритейла.

## Структура проекта
- `data/` - директория для хранения сгенерированных датасетов (CSV) и графиков.
- `generator/` - скрипты для генерации синтетических данных продаж.
- `model/` - директория для будущих моделей машинного обучения или алгоритмики расчета цен.
- `ui/` - директория для будущего пользовательского интерфейса.
- `notebooks/` - Jupyter ноутбуки для Exploratory Data Analysis (EDA) и экспериментов.

## Запуск
Для начала установите необходимые зависимости (важно убедиться, что установлен `matplotlib`, помимо `pandas` и `numpy`):

```bash
pip install pandas numpy matplotlib
```

Запустите скрипт генерации данных:

```bash
python generator/generate_data.py
```

После выполнения скрипта:
- В `data/sales_history.csv` сохранится датасет с историей продаж и цен.
- В `data/plots/` появятся 2 визуализации.
