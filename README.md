# Алгоритм динамического ценообразования для ритейла (MVP)

Данный проект представляет собой MVP-систему для автоматизации ценообразования в ритейле. Система объединяет генерацию синтетических данных, аналитические модели (эвристики и регрессию) и интерактивный интерфейс для управления ценами.

## 📁 Структура проекта

```text
Reteil/
├── model/
│   └── pricing.py         # ЦЕНТРАЛЬНАЯ ЛОГИКА (Единый источник правды)
├── generator/
│   └── generate_data.py   # Генератор данных с учетом конкурентов
├── ui/
│   └── streamlit_app.py   # Интерактивное приложение (Streamlit)
├── data/                  # Хранилище данных (CSV и графики)
│   └── sales_history.csv
├── requirements.txt       # Список зависимостей
└── .gitignore             # Исключение временных файлов и данных
```

## 🚀 Быстрый старт

1. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Сгенерируйте исторические данные**:
   ```bash
   python generator/generate_data.py
   ```

3. **Запустите веб-интерфейс**:
   ```bash
   streamlit run ui/streamlit_app.py
   ```

## ✨ Основные возможности

*   **Централизованная модель**: Вся логика расчетов (правила, регрессия $P = A/2B$, симуляция) вынесена в отдельное «ядро» `model/pricing.py`.
*   **Интерактивный Dashboard**:
    *   **Загрузка данных**: Возможность загрузить собственный CSV-файл через боковую панель.
    *   **Обзор**: Визуализация зависимости продаж от цены и динамики рынка.
    *   **Рекомендации**: Сравнение текущей цены с эвристической и оптимальной (регрессия).
    *   **Симуляция**: Прогноз выручки на 7-30 дней вперед при выборе различных стратегий.
*   **Реалистичный генератор**: Учитывает базовую эластичность спроса и демпфирование цен конкурентами.

## 🗓 Спринты проекта
1. **Спринт 1: Исследование и Генерация**. Создание базы для моделирования, разработка формул спроса (100 дней, 5 товаров).

## 🧠 Методология
### Эластичность спроса
Мы используем упрощенную линейную модель:
$Sales = A - B \cdot Price + \epsilon$
Где $B$ — коэффициент эластичности.

### Оптимизация цены
Функция выручки: $Rev = P \cdot (A - B \cdot P) = AP - BP^2$.
Максимум достигается в точке, где производная равна нулю:
$\frac{dRev}{dP} = A - 2BP = 0 \implies P_{opt} = \frac{A}{2B}$
